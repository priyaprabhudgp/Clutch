User Manual
Toast is a productivity game designed to help students stay on top of assignments while earning rewards. The app blends task management + gacha mechanics.
It allows users:
Track assignments
Earn coins → buy rewards
Customize a bread character
1. Create an account and login following page instructionsEnter:
2. The Dashboard is your main workspace.
Here you can: View assignments, Add new assignments, Submit assignments, See coin balance, View character preview
Your assignment appears in the list instantly after you fill the assignment form. You are rewarded more coins for submitting assignments early and punished if you have 4 or more overdue assignments!
Assignment Status Types → Assignments are categorized automatically:
Pending → Due in future
Overdue → Deadline passed
Completed → Coins earned
Character Preview (Dashboard)
A mini version of your bread appears on the Dashboard.
It reflects: Equipped items, Bread status
3.  Coin System
The earlier you submit your assignment, the more coins you earn. You need coins to buy the customizer packs!
4. Character System + Store
Your character is a customizable bread avatar.
You can visit the store to buy a item. When you click a pack, a random item from that pack will be purchased. If you are unlucky, you might get a dupe—in this case nothing new gets added to your inventory!
5. Bread Page
Navigate to Bread Page to customize. Here you can: View inventory and Equip items owned
Inventory Cupboard: contains items you’ve purchased in their gacha pack categories
Once you equip items, make sure to click “Save Customization button”
